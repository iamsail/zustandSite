# Project Brief

## Project Overview
`zustand.site` is a documentation and tutorial website for the Zustand state management library. The goal is to provide a comprehensive, SEO-friendly, and easy-to-navigate resource for developers learning and using Zustand.

## Core Requirements
- **Documentation:** Clear guides on how to use Zustand.
- **Tutorials:** Step-by-step tutorials for common use cases.
- **Examples:** Practical code examples.
- **SEO:** Optimized for search engines to attract relevant traffic.
- **i18n:** Multi-language support (English, Japanese, Chinese).

## Current Focus
Implementing a comprehensive SEO strategy based on traffic analysis, focusing on:
1.  **Comparison Content:** Intercepting traffic comparing Zustand with Redux/Context.
2.  **Advanced Features:** Deep diving into Persistence and Middleware.
3.  **Framework Integration:** Dedicated guides for Next.js (App Router/SSR).
4.  **TypeScript:** Improving visibility and CTR for TypeScript-related queries.
